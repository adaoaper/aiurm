# AIURM/AIUAR Protocol

**AIURM/AIUAR is a set of conventions and abstractions for persistent, governable, and expansible cognitive workflows.**

It creates an operational environment where data, logic, and results are organized, addressed, reused, and executed —
by any LLM, on any substrate, with no infrastructure dependency.

**Requires: LLM + CLI + filesystem — everyone already has this.**

---

> *"I need to define **what**, with **what**, and **for what**. The AI resolves the **how**."*

---

## AIUAR – AI Universal Address Reference [∞*]

AIUAR extends AIURM with a 5-level hierarchical addressing system:

```
aiuar_root/
└── *****contextspace
    └── ****entity
        └── ***project
            ├── governance/    ← pipeline contract
            └── **session
                ├── data/      ← *data_x
                ├── logic/     ← *logic_x
                └── result/    ← *result_x
```

One address. Any substrate. Zero ambiguity.
