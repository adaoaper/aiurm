# Purpose
End-to-end hit-to-lead optimization campaign with cognitive governance:
seed hit set -> analog expansion -> SAR-aware triage -> physicochemical repair -> structural alerts ->
ADMET / DMPK screen -> selectivity tiering -> synthesis feasibility -> building-block availability ->
retrosynthesis -> novelty / diversity -> uncertainty calibration -> policy reranking -> robustness ->
explainability.
