# Workflow
Execution Review (v0.2)
Per-Session Worker Reviewer for Any Target Project Execution

# Purpose
Generic reviewer that claims one completed session from any target AIUAR project at a time
and writes human-facing review artifacts directly INTO that same session, alongside the
target's own results. Stateless worker pattern; idempotent by claim on the absence of the
reviewer's final output marker (*result_preview_insights). Target project is chosen at
deployment time via aiuar_review_target (project-level address); aiuar_data_source and
aiuar_result_output are derived from it by field substitution. Completes the demo triad:
generator (project_cra_integration) -> executor (project_credit_risk_assessment) ->
reviewer (project_execution_review).
